# Tasheel BNPL Prototype

Dedicated Expo web prototype for the Tasheel BNPL flow from Figma node `1747:80160`.

Live site: https://hades898.github.io/tasheel-bnpl-prototype/

This project is separate from the Flash Cash prototype. It renders exported Figma frames directly for high visual fidelity and supports click/tap-through navigation across the BNPL flow.

## Run locally

```bash
npm install
npx expo start --web --port 8090
```

## Build

```bash
npx expo export --platform web
```
